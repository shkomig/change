# Change - אתר תדמית מקצועי

אתר תדמית מקצועי לעסק "change" המתמחה בפתרונות דיגיטליים מתקדמים.

## תכונות

- ✅ עיצוב מודרני ומקצועי
- ✅ תמיכה מלאה בעברית ובאנגלית
- ✅ תאימות מלאה למובייל
- ✅ אופטימיזציה לקידום בגוגל (SEO)
- ✅ אנימציות ואפקטים ויזואליים
- ✅ טופס יצירת קשר אינטראקטיבי
- ✅ קישורים לפרויקטים קיימים

## קבצים

- `index.html` - דף הבית בעברית
- `en.html` - דף הבית באנגלית
- `styles.css` - עיצוב האתר
- `script.js` - פונקציונליות JavaScript
- `logo.svg` - לוגו העסק
- `robots.txt` - הוראות לרובוטים של מנועי חיפוש
- `sitemap.xml` - מפת האתר לקידום

## פריסה

### 1. העלאה לשרת

העלו את כל הקבצים לשרת האינטרנט שלכם:

```
/
├── index.html
├── en.html
├── styles.css
├── script.js
├── logo.svg
├── robots.txt
└── sitemap.xml
```

### 2. הגדרת דומיין

הגדירו את הדומיין `change.co.il` (או דומיין אחר) להצביע לשרת.

### 3. הגדרת SSL

התקינו תעודת SSL להצפנת האתר (HTTPS).

### 4. קידום בגוגל

1. **Google Search Console**: הירשמו ל-Google Search Console והוסיפו את האתר
2. **Google Analytics**: הוסיפו Google Analytics למעקב אחר מבקרים
3. **Sitemap**: שלחו את `sitemap.xml` ל-Google Search Console
4. **Meta Tags**: כל המטא-תגים כבר מוגדרים בקוד

### 5. עדכון פרטי יצירת קשר

עדכנו את פרטי יצירת הקשר בקובץ `index.html` ו-`en.html`:

```html
<!-- עדכנו את הכתובת, הטלפון והאימייל -->
<span>info@change.co.il</span>
<span>+972-50-123-4567</span>
```

### 6. הגדרת טופס יצירת קשר

הגדירו שרת לטיפול בטופס יצירת הקשר או השתמשו בשירות כמו:

- Formspree
- Netlify Forms
- EmailJS
